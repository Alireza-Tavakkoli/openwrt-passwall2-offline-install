# آموزش نصب Passwall2 به صورت دستی روی OpenWrt
<br>

## مقدمه
در این آموزش یاد می‌گیرید چگونه Passwall2 را به صورت دستی (Manual) روی OpenWrt نصب کنید.

این روش برای زمانی است که:
- اینترنت آزاد بر روی روتر ندارید یا در کشوری زندگی میکنید که از همجا فیلتر است مثلا ایران
- یا می‌خواهید فایل‌ها را خودتان نصب کنید
- یا ابزار opkg / apk به درستی کار نمی‌کند
<br>

## ⚠️ حداقل مشخصات مورد نیاز

قبل از نصب، مطمئن شوید دستگاه شما توانایی اجرای Passwall2 را دارد:

### 🔹 رم (RAM)
- حداقل قابل اجرا: 128MB  
- پیشنهادی: 256MB یا بیشتر  
- کمتر از 128MB → احتمال لگ، کرش یا اجرا نشدن سرویس‌ها

### 🔹 پردازنده (CPU)
- حداقل پیشنهادی: 700MHz به بالا  
- بهتر: دو هسته‌ای یا 1GHz+

### 🔹 حافظه (Storage)
- حداقل: 16MB فضای خالی (لب مرز) که فعک نکنم بتونه جواب بده...  
- پیشنهادی: 32MB تا 128MB  
- چون Passwall2 حجم می‌گیرند

### 🔹 نسخه OpenWrt
- حداقل: OpenWrt 21.02  
- پیشنهادی: 23 / 24 / 25  
<br>

## 🔌 مرحله 1 : اتصال به روتر

قبل از هر کاری باید به روتر OpenWrt وصل شوید.

## 🟢 روش اول : PuTTY (ویندوز)

📥 دانلود:

👉 [https://soft98.ir/internet/network/3197-putty.html]


### 📌 مراحل اتصال

1. برنامه PuTTY را باز کنید
2. در قسمت **Host Name** این را وارد کنید:

```
192.168.1.1
```

3. روی **Open** کلیک کنید



### 🔑 ورود به روتر

وقتی صفحه سیاه باز شد:

* Username :

```
root
```

* Password :

```
رمز روتر (اگر تنظیم نکردی خالی بگذار)
```



## 🟡 روش دوم : Windows Terminal (جایگزین PuTTY)

اگر PuTTY نمی‌خواهید:

```bash id="sshwin"
ssh root@192.168.1.1
```

<br>

## 🍎 لینوکس / macOS

در ترمینال:

```bash id="sshmac"
ssh root@192.168.1.1
```
<br>

## ⚠️ نکات مهم

اگر وصل نشد:

- مطمئن شوید کامپیوتر به روتر وصل است
- آی‌پی ممکن است غیر از 192.168.1.1 باشد
- اولین اتصال ممکن است هشدار امنیتی بدهد (yes بزنید)

<br>

## 🎯 نتیجه این مرحله

اگر درست انجام شود باید وارد محیطی شوید که شبیه این است:

```
root@OpenWrt:~#
```
<br>

## 📊 مرحله 2 : پیدا کردن نسخه OpenWrt و معماری CPU

قبل از دانلود Passwall2 باید دقیقاً بدانید:

* نسخه OpenWrt
* معماری CPU دستگاه



## 🧠 1. بررسی نسخه OpenWrt

وارد SSH شوید و این دستور را بزنید:

```bash id="v1"
cat /etc/openwrt_release
```


### 📌 خروجی نمونه:

```id="v2"
DISTRIB_RELEASE='23.05.2'
DISTRIB_ARCH='aarch64_cortex-a53'
```



## 📌 معنی موارد مهم:

* `DISTRIB_RELEASE` →  OpenWrt نسخه
* `DISTRIB_ARCH` →  CPU معماری 


اگر در یاداشت معماری پردازنده و نسخه OpenWrt اشتباه کنید:

* ❌ پکیج نصب نمی‌شود
* ❌ یا ارور architecture می‌دهد
* ❌ یا Passwall اجرا نمی‌شود
<br>


## 📥 مرحله 3 : دانلود فایل‌های مناسب Passwall2
برای اینکار نیاز به فیلتر شکن دارید تا سایت بالا بیاید .
<br>

## 🌐 سایت دانلود

به صفحه زیر بروید:

```text
https://sourceforge.net/projects/openwrt-passwall-build/files/
```

##  🔑 دانلود کلید ها 
اگر OpenWrt نسخه 24 به پایین دارید :


```text
ipk.pub
```
را [دانلود](https://sourceforge.net/projects/openwrt-passwall-build/files/ipk.pub/download) کنید. 
<br>

و اگر OpenWrt نسخه 25 به پایین دارید :


```text
apk.pub
```
را [دانلود](https://sourceforge.net/projects/openwrt-passwall-build/files/apk.pub/download) کنید. 

## 📂 انتخاب نسخه صحیح

به پوشه releases بروید و packages متناسب با نسخه OpenWrt خود انخاب کنید مثلا اکر نسخه 24.10 بود باید بر روی پوشه   packages-24.10 کلیک کنید

سپس معماری مناسب دستگاه خود را انتخاب کنید. برای مثال اگر arm_cortex-a15_neon-vfpv4 بود بر روی پوشه arm_cortex-a15_neon-vfpv4 کلیک کنید 

الان به دو پوشه passwall_packages و passwall_luci کار داریم
اول بر روی passwall_packages کلیک کرده  

دراینجا اگر مشکل حجمی ندارید تمام فایل ipk یا apk را دانلود کنید (و اگه مشکل حجمی دارید حال ندارم لیست چیزای ضروری بنویسم پس ارزو موفقیت میکنم برات) 
و بعد به قسمت passwall_luci برید و luci-app-passwall* دانلود کنید 


## ⚠️ قبل از دانلود

حتماً مطمئن شوید:

✅ نسخه OpenWrt فایل با نسخه روتر شما یکی باشد.

✅ معماری فایل با معماری دستگاه شما یکی باشد.

در غیر این صورت با خطاهایی مثل:

```text
incompatible with the architectures
```

یا

```text
cannot satisfy dependencies
```

مواجه خواهید شد.


## 📤 مرحله 4 : انتقال فایل‌ها به روتر

بعد از دانلود فایل‌های Passwall2 باید آن‌ها را به روتر OpenWrt منتقل کنید.



## 🟢 روش 1: WinSCP (ویندوز - ساده‌ترین روش)

### 📥 دانلود:

👉 [https://soft98.ir/internet/ftp-tools/748-winscp.html]


### 📌 مراحل اتصال:

1. برنامه WinSCP را باز کنید
2. اطلاعات زیر را وارد کنید:

* File protocol: `SCP`
* Host name: `192.168.1.1`
* Username: `root`
* Password: رمز روتر

3. روی Login کلیک کنید

---

### 📂 انتقال فایل‌ها:

فایل‌های دانلود شده را به این مسیر منتقل کنید:

```bash id="tmp3"
/tmp/
```

---

## 🟡 روش 2: SCP (لینوکس / macOS / Windows Terminal)

اگر ترمینال دارید:

```bash id="scp3"
scp passwall2.ipk root@192.168.1.1:/tmp/
```

اگر چند فایل دارید:

```bash id="scp4"
scp *.ipk root@192.168.1.1:/tmp/
```

---

## 🍎 روش 3: Linux / macOS ساده

```bash id="scp5"
scp file.ipk root@192.168.1.1:/tmp/
```

---

## ⚠️ نکات مهم

* همه فایل‌ها باید داخل `/tmp/` باشند
* اگر چند فایل دارید، همه را منتقل کنید
* ترتیب مهم است (اول dependency ها بعد main package)

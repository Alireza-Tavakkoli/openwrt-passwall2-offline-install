# آموزش نصب Passwall2 به صورت دستی روی OpenWrt

## مقدمه
در این آموزش یاد می‌گیرید چگونه Passwall2 را به صورت دستی (Manual) روی OpenWrt نصب کنید.

این روش برای زمانی است که:
- اینترنت آزاد بر روی روتر ندارید یا در کشوری زندگی میکنید که محدود است مثلا ایران
- یا می‌خواهید فایل‌ها را خودتان نصب کنید
- یا ابزار opkg / apk به درستی کار نمی‌کند

---
## ⚠️ حداقل مشخصات مورد نیاز

قبل از نصب، مطمئن شوید دستگاه شما توانایی اجرای Passwall2 را دارد:

### 🔹 رم (RAM)
- حداقل قابل اجرا: 128MB  
- پیشنهادی: 256MB یا بیشتر  
- کمتر از 128MB → احتمال لگ، کرش یا اجرا نشدن سرویس‌ها

---

### 🔹 پردازنده (CPU)
- حداقل پیشنهادی: 700MHz به بالا  
- بهتر: دو هسته‌ای یا 1GHz+  
- برای استفاده سنگین (چند دستگاه همزمان) → CPU قوی‌تر لازم است

---

### 🔹 حافظه (Storage)
- حداقل: 16MB فضای خالی (لب مرز)  
- پیشنهادی: 32MB تا 128MB  
- چون Passwall2 + وابستگی‌ها حجم می‌گیرند

---

### 🔹 نسخه OpenWrt
- حداقل: OpenWrt 21.02  
- پیشنهادی: 23 / 24 / 25

- dependency error
- installation failed
